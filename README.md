# 🌱 STM32 Automatic Plant Irrigation System

A compact automatic irrigation system using **STM32F103C8T6**, **soil moisture sensor**, **RTC DS1307**, **LCD 1602**, and **water pump**.  
Includes firmware Keil C, Proteus simulation, hardware schematic, and project documentation.

---

## 🚀 Features
- Automatic watering based on soil moisture  
- mosfet-driven 12V pump control  
- LCD 1602   
- Proteus simulation + custom hardware design  

---

## 🧩 System Overview
- **MCU:** STM32F103C8T6  
- **Sensors:** Soil Moisture SMS-V1  
- **Display:** LCD1602   
- **Actuator:** mosfet_opto + Pump  
- **Design:** Schematic, PCB, Proteus simulation
